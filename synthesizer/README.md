# synthesizer

## 定位
负责把多子任务证据与中间总结整合为结构化、可引用的最终研究报告。

## 负责内容
- 生成子任务级总结（局部结论与证据归纳）。
- 基于所有子任务总结生成全局大纲。
- 按大纲整合并产出最终报告。
- 维护引用关系与可追溯性，确保结论可回链到证据。

## 总体实现思路
- 采用“先局部后全局”的分层合成流程，控制信息丢失与逻辑断裂。
- 报告结构固定化（背景、方法、发现、证据、结论、不确定性、下一步建议）提升稳定性。
- 对冲突证据进行显式标注，不强行合并为单一结论。
- 输出时兼顾可读性与可验证性，保证研究结果可复核。

## 边界约束
- 不做检索执行，不做流程调度。
- 只负责“把已有证据组织成高质量可交付内容”。
